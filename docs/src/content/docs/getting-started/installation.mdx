---
title: Installation
description: How to install Pipe on your machine.
---

## Homebrew

```bash
brew tap getpipe-dev/pipe
brew install pipe
```

## GitHub Releases

Download the latest binary for your platform from the [Releases](https://github.com/getpipe-dev/pipe/releases) page, extract it, and move it to a directory in your `PATH`.

```bash
# Example for Linux amd64
curl -Lo pipe.tar.gz https://github.com/getpipe-dev/pipe/releases/latest/download/pipe_linux_amd64.tar.gz
tar xzf pipe.tar.gz
sudo mv pipe /usr/local/bin/
```

## Verify

```bash
pipe --version
```
