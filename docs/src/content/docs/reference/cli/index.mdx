## Core Commands

| Command | Description |
|---------|-------------|
| [`pipe <pipeline>`](/reference/cli/run/) | Run a pipeline |
| [`pipe init`](/reference/cli/init/) | Create a new pipeline |
| [`pipe list`](/reference/cli/list/) | List all pipelines |
| [`pipe lint`](/reference/cli/lint/) | Lint a pipeline for errors and warnings |
| [`pipe inspect`](/reference/cli/inspect/) | Show detailed pipeline metadata |
| [`pipe rm`](/reference/cli/rm/) | Remove a pipeline |
| [`pipe cache`](/reference/cli/cache/) | Manage step cache entries |
| [`pipe alias`](/reference/cli/alias/) | Manage pipeline aliases |

## Hub Commands (Beta)

| Command | Description |
|---------|-------------|
| [`pipe login`](/reference/cli/login/) | Authenticate with Pipe Hub |
| [`pipe logout`](/reference/cli/logout/) | Log out of Pipe Hub |
| [`pipe pull`](/reference/cli/pull/) | Pull a pipeline from Pipe Hub |
| [`pipe push`](/reference/cli/push/) | Push a pipeline to Pipe Hub |
| [`pipe mv`](/reference/cli/mv/) | Move a local pipeline to Pipe Hub |
| [`pipe switch`](/reference/cli/switch/) | Switch the active tag for a hub pipeline |
| [`pipe tag`](/reference/cli/tag/) | Manage tags for a hub pipeline |

## Global flags

| Flag | Description |
|------|-------------|
| `-v`, `--verbose` | Increase output verbosity (`-v` verbose, `-vv` debug) |
| `--version` | Print version and exit |
| `-h`, `--help` | Show help |
